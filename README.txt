
本固件（非Lite2版） 自带 负载均衡/MWAN3分流助手/IPTV助手，但他们都是默认关闭的，如果你需要启用他们，需要到 “系统”--“启动项”里找到。
他们分别是
负责均衡 -- mwan3
MWAN3分流助手 -- mwan3helper
IPTV助手 -- iptvhelper
将其改成已启用，然后重启。

支持网页上传刷机和网络自检

命令行刷机
wget ROM下载地址 -O /tmp/rom.zip
unzip rom.zip
sh /root/flash_rom.sh /tmp/rom具体取名.img.gz /dev/mmcblk0 gz
